# How-to

## How to install
* Start by making a backup (both DB and store files)!!!
* Admin > Extensions > Installer > [Upload]: customer-group-size.ocmod.zip
* Admin > Extensions > Modifications > [Refresh]
* Admin > Dashboard > [Developer Settings] > [Refresh]
* Admin > Extensions > Extensions > Modules > [underr] Customer Group Size > [Install]

## How to uninstall
* Start by making a backup (both DB and store files)!!!
* Admin > Extensions > Extensions > Modules > [underr] Customer Group Size > [Uninstall]
* Admin > Extensions > Installer > Install History > customer-group-size.ocmod.zip > [Uninstall]
* Admin > Extensions > Modifications > [Refresh]
* Admin > Dashboard > [Developer Settings] > [Refresh]

## How to update
